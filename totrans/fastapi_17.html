Part IV. A Gallery

In Part III, you built a minimal website with some basic code. Now let’s do something fun with it. The following chapters apply FastAPI to common web uses: forms, files, databases, charts and graphics, maps, and games.

To tie these applications together and make them more interesting than the usual dry computing book examples, we’ll plunder data from an unusual source, some of which you’ve already glimpsed: imaginary creatures from world folklore and the explorers who pursue them. There will be yetis, but also more obscure—though no less striking—members.
